FlowFix is a Chrome extension that helps you diagnose and optimize your Zapier and Make.com workflows. This privacy policy explains how FlowFix handles your data.

Our Core Promise

FlowFix respects your privacy.

We do not collect, store, or share your personal data. All analysis happens locally on your device.

What FlowFix Does

FlowFix analyzes the DOM (visual structure) of your Zapier and Make.com dashboards to extract workflow metrics:

  • Zapier: Task usage, task limits, zap counts, error indicators
  • Make.com: Operation usage, operation limits, scenario counts, status indicators

These metrics are displayed in a badge on your dashboard and a popup for quick reference.

What We Do NOT Do

  • ‚ùå Send your data to external servers
  • ‚ùå Collect personal information beyond what's visible in the DOM
  • ‚ùå Use analytics or tracking services
  • ‚ùå Share your workflow names, email, or account details
  • ‚ùå Make network requests (except to Zapier/Make sites you're already viewing)
  • ‚ùå Store session tokens or authentication credentials

Local Storage

FlowFix stores the following information only on your device using Chrome's chrome.storage.local:

  • Extracted metrics (task/operation counts, health scores)
  • Timestamp of when metrics were last collected
  • Current page URL (without query parameters or sensitive data)
  • Team name and plan name (as visible on the dashboard)

This data is never transmitted over the network. It remains in your browser's local storage until you clear it manually.

Permissions Explained

FlowFix requests only the minimum permissions needed to function:

  • activeTab
    Allows FlowFix to analyze the current page's DOM when you click the extension icon or when it detects a Zapier/Make dashboard.
  • storage
    Allows FlowFix to save extracted metrics locally for display in the popup.

We explicitly do NOT request:

  • webRequest ‚Äî We don't intercept or analyze your network traffic
  • tabs ‚Äî We only access the active tab, not your browsing history
  • <all_urls> ‚Äî We only work on zapier.com and make.com

Host Permissions

FlowFix operates only on these websites:

  • zapier.com/*
  • www.make.com/*
  • us1.make.com/*
  • eu1.make.com/*
  • eu2.make.com/*

On these sites, FlowFix can only read the page DOM‚Äîit cannot modify your workflows or settings.

Security

  • FlowFix uses Shadow DOM to prevent page CSS/JavaScript from interfering with the extension
  • All DOM element creation uses safe APIs (no innerHTML)
  • No third-party libraries are loaded from external CDNs
  • No external API calls are made except to Zapier/Make (which you're already connected to)

Deleting Your Data

You have complete control over data stored by FlowFix:

  • Clear extension storage: Go to chrome://apps, find FlowFix, right-click ‚Üí Delete ‚Üí Remove
  • Keep extension, clear data: The "Delete Extension Data" option in Chrome's extension page
  • Manual browser storage: Chrome DevTools ‚Üí Application ‚Üí Storage ‚Üí Local Storage

If we update this privacy policy, we'll notify you by updating the "Last Updated" date. Significant changes will be announced when you update to a new version of FlowFix.
